# Clone-Hero-Overlay
A simple python script that creates an overlay for Guitar Hero / CLone Hero / Rock Band / YARG / Fortnite Festival (PC) or any other rhythm game on PC to show each fret, strum, and whammy as it's activated. Includes a counter for each fret, as well as a reset. Source code and compiled file included.


Run in terminal via python ch_overlay.py

Right click on the overlay to close it.
